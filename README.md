# Tic Tac Toe Game

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript.

This project was created to practice JavaScript concepts like DOM manipulation, event handling, game logic, conditional statements, and dynamic UI updates.

---

## Features

* Interactive Tic Tac Toe gameplay
* Player symbol selection (X or O)
* Dynamic board updates
* Winner detection
* Reset game functionality
* Hover effects and animations
* Clean and responsive UI
* Centered game layout

---

## Tech Stack

* HTML
* CSS
* JavaScript

---

## Concepts Practiced

* DOM Manipulation
* Event Listeners
* Conditional Statements
* Game Logic
* Loops
* Dynamic UI Updates
* Hover Effects
* Flexbox

---

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser
